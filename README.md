# 🌍 Global Currency Converter

A Java-based currency converter developed using Object-Oriented Programming (OOP) concepts and static members. The application converts USD, EUR, and GBP into INR using predefined exchange rates without requiring object creation.

## Features

* USD to INR conversion
* EUR to INR conversion
* GBP to INR conversion
* Static variables and methods
* Utility class design
* Simple console-based interface

## Technologies

* Java
* OOP
* Static Members
